# Fractal Geometry Visualizer: Iterated Function Systems (IFS)

## 🌿 The Story Behind the Name: GoldenFern

The name **GoldenFern** and my GitHub avatar are directly derived from this project. The image `IFS.png` (the golden fern) is generated by the `main` block of the `IFS.py` script. 

While the classic Barnsley Fern is a well-known fractal, the version used for my avatar is a customized variation. By tweaking the affine transformation parameters and introducing a sine-based non-linearity in the leaflets, I created a more organic, "golden" curvature that represents the intersection of mathematics and nature.

![Golden Fern Avatar](IFS.png)

## 📖 Introduction

This repository provides a Python-based framework for visualizing 2D Fractal Geometry using **Iterated Function Systems (IFS)**. It implements the **Chaos Game** algorithm to render complex attractors from simple sets of contraction mappings.

### Features:

- **High-Resolution Rendering**: Uses `matplotlib` to export high-DPI static images.
- **Interactive Exploration**: Uses `plotly` with WebGL (`Scattergl`) for real-time zooming and panning of fractals with millions of points.
- **Predefined Fractals**: Includes definitions for the Barnsley Fern, Sierpinski Triangle, Koch Curve, Dragon Curve, Lévy C Curve, and more.
- **Random Fractal Generator**: A module to procedurally generate unique, mathematically valid contraction mappings.

## 🚀 Getting Started

### Prerequisites

Ensure you have Python 3.8+ installed. Install the required libraries:

```bash
pip install numpy matplotlib plotly
```

### Usage

1. **Generate the Avatar (Golden Fern):**
   Simply run the script as is:
   
   ```bash
   python IFS.py
   ```
2. **Generate Other Fractals:**
   Modify the `if __name__ == "__main__":` block to use different definitions:
   
   ```python
   # Example: Generate a Dragon Curve
   visualizer = IFSVisualizer(*define_dragon_curve())
   visualizer.generate_points(n_total=500000, m_discard=1000)
   visualizer.plot_matplotlib(save=True, color="blue")
   ```

## 🧮 Mathematical Principles

### 1. Iterated Function Systems (IFS)

An Iterated Function System is a collection of a finite number of contraction mappings $f_1, f_2, \dots, f_n$ on a complete metric space $\mathbb{X}$. In this project, we focus on affine transformations in $\mathbb{R}^2$:
$$ f_i(x) = A_i x + b_i $$
where $A_i$ is a $2 \times 2$ matrix and $b_i$ is a translation vector. For the system to be "contractive," the Lipschitz constant of each $f_i$ must be less than 1.

According to **Hutchinson's Theorem**, there exists a unique non-empty compact set $A \subset \mathbb{X}$, called the **attractor**, such that:
$$ A = \bigcup_{i=1}^{n} f_i(A) $$
The **Chaos Game** algorithm used in this script approximates this set $A$ by picking a random starting point $x_0$ and iteratively applying $x_{k+1} = f_{\sigma_k}(x_k)$, where $\sigma_k$ is chosen randomly from $\{1, \dots, n\}$ according to a probability distribution $p_i$.

### 2. Hausdorff Measure and Dimension

Standard Euclidean geometry (length, area, volume) fails to describe the complexity of fractals. Instead, we use the **Hausdorff Measure**.

For a set $F \subset \mathbb{R}^n$ and a real number $s \ge 0$, the $s$-dimensional Hausdorff measure $\mathcal{H}^s(F)$ is defined as:
$$ \mathcal{H}^s(F) = \lim_{\delta \to 0} \inf \left\{ \sum_{i=1}^{\infty} |U_i|^s : F \subset \bigcup_{i=1}^{\infty} U_i, |U_i| < \delta \right\} $$
where $|U_i|$ denotes the diameter of the set $U_i$. 

The **Hausdorff Dimension** $d_H(F)$ is the unique value $s$ at which the measure $\mathcal{H}^s(F)$ jumps from $\infty$ to $0$:
$$ d_H(F) = \inf\{s : \mathcal{H}^s(F) = 0\} = \sup\{s : \mathcal{H}^s(F) = \infty\} $$
For many IFS fractals (like the Sierpinski Triangle), the dimension is a non-integer (e.g., $\log 3 / \log 2 \approx 1.585$), which quantifies how the fractal fills space more effectively than a line but less so than a plane.
